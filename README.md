■ 概要

このアプリは、ボタンを押すと高速で画像が切り替わり、

停止後におみくじ結果が表示される「スロット式おみくじアプリ」です。

アニメーションによる臨場感とシンプルな操作性で、誰でも気軽に楽しめます。



■　使用技術

・JavaScript

・HTML/CSS



■　主な機能

・スロットのように高速で画像が切り替わる演出

・何度でも引き直し可能

・簡単なメッセージを表示。



■　起動方法

・インストール作業やコマンド操作は不要です。

・このリポジトリをダウンロード or クローン

・index.html` をダブルクリックしてブラウザで開くだけで実行可能！
